CORE CLASSES & METHODS
================================================================================

TechTree Class Methods:

- register_technology(tech: Technology) -> None
  Register a new technology in the tree

- get_available_techs(player_id: str) -> List[Technology]
  Get all technologies ready for research

- start_research(player_id: str, tech_id: str) -> Tuple[bool, str, ResearchProject]
  Begin researching a technology

- advance_research(player_id: str, project_id: str, points: int) -> Tuple[bool, str, float]
  Invest research points in active project

- complete_research(player_id: str, project_id: str) -> Tuple[bool, str, float]
  Finalize research and apply bonuses

- pause_research(project_id: str) -> Tuple[bool, str]
  Pause research temporarily

- resume_research(project_id: str) -> Tuple[bool, str]
  Resume paused research

- get_tech_by_era(era: Era) -> List[Technology]
  Filter technologies by era

- get_tech_by_philosophy(philosophy: ResearchPhilosophy) -> List[Technology]
  Filter technologies by research philosophy

- get_research_tree() -> Dict[str, Any]
  Get complete tree structure for visualization

- get_research_report(player_id: str) -> Dict[str, Any]
  Get comprehensive research status for a player

- is_tech_completed(player_id: str, tech_id: str) -> bool
  Check if player has completed a tech

- get_unlocked_by_tech(tech_id: str) -> Dict[str, List[str]]
  Get what a technology unlocks

================================================================================
GAMEPLAY INTEGRATION
================================================================================

RESEARCH FLOW:
1. Player allocates research_points per turn
2. Points advance active research project
3. Progress accumulates toward research_cost
4. Random breakthrough chance provides speed boost
5. Upon completion, bonuses apply and unlocks trigger

UNLOCK CASCADE:
1. Technology Complete
2. New Technologies Unlock (prerequisites satisfied)
3. Quests Unlock (quest_ids become available)
4. Perks Unlock (gameplay bonuses activate)
5. Features Unlock (new systems become playable)
6. Player Chooses Next Research Path

BONUS SYSTEM:
- Military: military_strength, armor, defense
- Scientific: research_speed, automation, breakthrough
- Cultural: morale, cultural influence, trade profit
- Consciousness: stability, collective_consciousness, transcendence
