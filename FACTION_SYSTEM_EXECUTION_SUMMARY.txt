## Key System Features

### Reputation System
- Continuous scale: 0.0 → 1.0
- 5 automatic unlock tiers: 20%, 40%, 60%, 80%, 95%
- Perks unlock automatically at thresholds
- Quests available based on progression
- No faction locks (can switch anytime)

### Perk System
- 40 unique perks (5 per faction)
- Automatic unlocking at reputation thresholds
- 14 bonus types affecting gameplay
- Permanent effects (no duration timers for most)
- Stackable with other bonuses

### Quest System
- 24 total quests (3 per faction)
- Difficulty scaling: Easy, Hard, Legendary
- Resource + reputation rewards
- Prerequisites and unlocking logic
- Turn availability windows

## API Quick Start

```python
# Initialize
from federation_game_factions import build_faction_system
system = build_faction_system()

# Player joins faction
system.join_faction("player_001", "diplomatic_corps")

# Gain reputation
system.change_reputation("player_001", "diplomatic_corps", 0.15)

# Get active perks
perks = system.get_faction_perks("diplomatic_corps", "player_001")
for perk in perks:
    print(f"{perk.perk_name}: +{perk.bonus_value}")

# Available quests
quests = system.get_faction_missions("diplomatic_corps", "player_001")
for quest in quests:
    print(f"{quest.quest_name} - {quest.difficulty}")

# Complete quest
success, rewards = system.complete_quest(
    "player_001",
    "diplomatic_corps",
    "first_treaty"
)

# Get bonuses for gameplay
faction = system.get_player_faction("player_001")
bonuses = faction.get_faction_bonuses("player_001")
for bonus_type, value in bonuses.items():
    # Apply to game calculations
    gameplay_value *= (1.0 + value)

# Save data
data = system.export_faction_data()
# Save to database/file
```
