If you have tried twice and cannot solve a problem: STOP. Do not try a third time. Instead, generate a PROJECT SUMMARY BRIEF using this exact format:

```

PROJECT: [name]
STACK: [languages, frameworks, tools]
```

Do not include code blocks unless essential. Do not include error logs. Write it so someone who cannot see the project understands it completely in 30 seconds. This brief will be taken to external AI for resolution. Your job is to make the brief perfect, not to keep guessing.
